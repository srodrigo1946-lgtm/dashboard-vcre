# VCRED Dashboard 2026

Dashboard interativo de análises de crédito imobiliário — Novolar / Esquadrão Kayser.

## Acessar

Abra `index.html` em qualquer navegador moderno (Chrome, Edge, Firefox).

## Recursos

- 📊 KPIs interativos (clique pra filtrar)
- 🏆 Pódio e ranking por mês e ano
- 📂 Carregar nova planilha xlsx
- ✏️ Cadastrar vendas manualmente
- 📸 Fotos dos gerentes
- 🎨 Temas (Clássico, Champions, Senna, Olímpico, F1, Neon)
- 💾 Auto-save no navegador
- 📥 Backup/Restore

## Standalone

Arquivo único `index.html` — funciona offline, sem servidor, sem internet.
